1. 購買插件後, 在 [Discord取貨頻道](https://discord.com/channels/1166519479379963924/1308535596477583420) 獲取插件並將其安裝到 `./plugins` 中  
2. 重啟伺服器  
3. Enjoy  

:::danger

請避免使用 `/reload` 或類似於 `PlugMan` 的插件進行**熱重載**, 否則插件將無法重新加載

:::
