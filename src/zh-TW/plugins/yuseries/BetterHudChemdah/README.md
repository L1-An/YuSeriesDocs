> 支援版本: 1.19 - 1.20.4

BetterHudChemdah 是一款基於 `BetterHud` 與 `Chemdah` 實現的**擴展對話主題**插件

## 特色

- 告別 Chemdah 原版的聊天框對話，使用更先進的 BetterHud 顯示在玩家的介面上
- 新增支援 Chemdah 的任務追蹤在 BetterHud 的指南針上
- 支援給每個 NPC 設置對話頭像，甚至支援 NPC 的表情變化！
- 高度客製化
- 沈浸的對話鏡頭系統
