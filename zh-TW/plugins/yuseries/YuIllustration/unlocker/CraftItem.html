Craft Item

當玩家製作原版物品時觸發

id

craft item

Condition

item

  • 類型: string
  • 默認值: none
  • 描述: 製作的物品類型。查看所有物品類型名稱 here, 注意必須小寫
Goal

amount 可選

  • 類型: number
  • 默認值: 1
  • 描述: 製作的物品數量
Example
unlock:
  - type: "craft item"
    condition:
      item: "iron_sword"
    goal:
      amount: 5
