Pickup Item

Pickup Item

當玩家撿起物品時觸發

id

pickup item

Data

重要

你需要設置至少一個以下數據.

name 可選

  • 類型: string
  • 默認值: none
  • 描述: 物品的名稱。

lore 可選

  • 類型: string
  • 默認值: none
  • 描述: 物品的描述。

nbt 可選

  • 類型: string
  • 默認值: none
  • 描述: 物品的 NBT。
Condition

yui-item

id: yui-item / item

語法:

  • item take 從玩家背包中移除撿起的物品。
  • item total-amount 返回撿起的物品總數。
  • item pick-amount 返回本次撿起的物品數量。
Variable

type

返回撿起的物品類型名稱

display-name

返回撿起的物品顯示名稱

Example
- type: "pickup item"
  data:
    name: "diamond" # 物品名稱
    lore: "diamond" # 物品描述（只要描述中包含對應的內容）
    nbt: "diamond" # 物品 NBT，默認包含 @yuillustration-item-pickup
  condition: |-
    check yui-item total-amount == 5
