    YuSeries Plugins

    YuIllustration

    A powerful illustration plugin, but not only for illustrations, unlimited categories, rich unlock conditions, and highly customizable.

    YuSpawnerHologram

    Add holograms to display the respawn time for your MythicMobsSpawner.

    YuBattleMusic

    Add battle music to your server, support single monster, global music, and highly customizable.

    YuItemAction

    A lightweight and simple item execution script action plugin.

    YuVarieLevel

    Add variable level system to your server, support multiple levels, custom experience value formula, and script when upgrading.

    Sortilege

    A powerful attribute plugin, multiple built-in attributes, and support custom attributes.

    BetterHudChemdah

    A plugin that extends the dialogue theme of Chemdah using BetterHud.
