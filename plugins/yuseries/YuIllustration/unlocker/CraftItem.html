Craft Item

This unlocker will trigger when a player crafts a vanilla item.

id

craft item

Data

type Required

  • Type: string
  • Default: none
  • Description: The type of the item crafted. See all item type names here.
Condition

craft-item

id: yui-craft-item / craft-item

Syntax:

  • craft-item total-amount returns the total number of the crafted item.
Example
- type: "craft item"
  data:
    type: "IRON_SWORD"
  condition: |-
    check craft-item total-amount == 5
